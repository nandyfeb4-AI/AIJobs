# AIJobs

Monorepo for the AIJobs platform.

## Apps

- `apps/web`: Next.js frontend
- `apps/api`: NestJS API
- `apps/worker`: background workers

## Packages

- `packages/types`: shared TypeScript types
- `packages/utils`: shared utilities
- `packages/config`: shared config helpers

## Commands

```bash
npm install
npm run dev:web
npm run dev:api
npm run dev:worker
npm run build
npm run typecheck
```
